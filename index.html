Guide
  • Type: Primary, Military, etc.
  • Price: only for chosen upgrade, not counting all the previous tiers
  • Ability: does this upgrade have an ability
  • Paragon: does this tower have a paragon
  • Tier: 3/4/5, tier 1 and 2 upgrades aren't available
  • Damage: damage types of a tower, like normal, camo, plasma, shatter etc. (only up to 3 show here, but it can have more damage types, for example, sentry expert)
  • Path: top/middle/bottom
